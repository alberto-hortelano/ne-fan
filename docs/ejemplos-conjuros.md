# Ejemplos de Conjuros

Guía de referencia para calibrar la potencia y los efectos de los conjuros.

Recordar: los puntos siguen la progresión 0, 1, 2, 3, 5, 8 (niveles 0–5).

---

## 1. La rata de la mazmorra

**Situación**: Un mago encerrado en una mazmorra, sin recursos, necesita escapar.

| Componente | Nivel | Puntos | Descripción |
|------------|-------|--------|-------------|
| Momento    | 2     | 2      | Espera 6 meses hasta una noche concreta |
| Ritual     | 1     | 1      | Marcas trazadas en el suelo de tierra |
| Objeto     | 0     | 0      | No tiene acceso a ningún objeto |
| Lugar      | 0     | 0      | La propia mazmorra, sin relevancia mágica |
| Habilidad (Hechizo) | 2 | 2 | Nivel 2 de 5 |
| **Potencia total** | | **5** | |
| **Duración** | 1 | −1 | Fugaz: lo justo para que la rata traiga las llaves |
| **Potencia efectiva** | | **4** | |

**Efecto**: Hechiza a una rata para que le acerque las llaves de la celda mientras el guardia duerme. La rata obedece una orden simple y concreta durante un breve periodo.

**Notas de diseño**: Potencia 5 basta para un control mental básico sobre un animal pequeño, con una tarea sencilla y de corta duración. El ingenio del mago compensa su falta de recursos. Al estar hecho casi solo con habilidad, un desencanto apenas tendría consecuencias para el mago, pero en la práctica solo se podría deshacer mientras dure el efecto.

---

## 2. La espada del naufragio

**Situación**: Un mago maestro quiere forjar un arma legendaria vinculada a la historia del rey Absolon, cuyo barco naufragó en batalla hace casi un siglo.

| Componente | Nivel | Puntos | Descripción |
|------------|-------|--------|-------------|
| Momento    | 4     | 5      | El centenario exacto del naufragio. Difícil de averiguar con precisión, 5 años tras iniciar la preparación del conjuro |
| Ritual     | 3     | 3      | Forja de la espada durante un mes completo |
| Objeto     | 5     | 8      | La empuñadura de la espada del rey Absolon, perdida en el fondo del mar durante años tras el naufragio. Recuperarla requiere varios conjuros previos |
| Lugar      | 3     | 3      | El cráter de un volcán |
| Habilidad (Artificio) | 5 | 8 | Nivel máximo |
| **Potencia total** | | **27** | |
| **Duración** | 5 | −8 | Permanente: la espada conserva su poder para siempre |
| **Potencia efectiva** | | **19** | |

**Efecto**: Cada ataque con la espada, aunque se pare o se esquive, salpica con agua a la víctima provocando asfixia. Penalización de -1 a todas las habilidades, acumulativo con cada golpe, hasta que la víctima se desmaye.

**Notas de diseño**: Potencia 27 es un conjuro de altísimo nivel. Requirió años de preparación y varios conjuros previos solo para recuperar el objeto de nivel 5. El mago necesita habilidad 5 para poder usar componentes de ese nivel. Un desencanto tendría consecuencias gravísimas para el mago dada la cantidad de potencia invertida en componentes.

## 3. El brazo del roble

**Situación**: Un guerrero ha perdido el brazo en batalla. Una curandera con conocimientos de magia antigua decide injertarle una rama viva de un viejo roble para que le sirva de brazo.

| Componente | Nivel | Puntos | Descripción |
|------------|-------|--------|-------------|
| Momento    | 2     | 2      | El equinoccio de primavera, cuando la vida se renueva |
| Ritual     | 3     | 3      | Una semana de cantos y cataplasmas de hierbas, alimentando la rama con su propia sangre |
| Objeto     | 2     | 2      | Una rama viva de un roble centenario |
| Lugar      | 0     | 0      | Junto al lecho del guerrero |
| Habilidad (Vida) | 3 | 3 | Nivel 3 de 5 |
| **Potencia total** | | **10** | |
| **Duración** | 3 | −3 | Duradero: meses, una estación |
| **Potencia efectiva** | | **7** | |

**Efecto**: La rama se funde con el muñón y se convierte en un brazo de madera viva. Funciona como un brazo real: tiene fuerza, sensibilidad y obedece la voluntad del guerrero. En primavera le brotan hojas. Es ligeramente más fuerte que un brazo de carne pero no siente dolor, lo que puede ser tanto ventaja como peligro.

**Mantenimiento**: Cada estación, la curandera prepara una cataplasma de savia y hierbas y la aplica al hombro donde se une la madera a la carne, recitando unas palabras. Si deja de hacerlo, el brazo se seca y se vuelve leña muerta en pocas semanas.

**Notas de diseño**: Potencia efectiva 7 permite una transformación significativa pero dentro de los límites de lo natural: no crea materia de la nada, sino que adapta algo vivo. El mantenimiento estacional encaja narrativamente — el brazo es madera viva y necesita cuidados como una planta. Esto crea dependencia entre el guerrero y la curandera, lo cual es interesante para la historia. Un desencanto le causaría una marca permanente a la curandera (nivel medio de componentes).
