# Efectos y Costes

Sistema genérico de efectos. La IA del motor de historia da el toque narrativo (la "asfixia" de la espada es mecánicamente daño + penalización con aplicación al atacar). Los costes se restan de la potencia efectiva.

---

## Efectos base

El núcleo de lo que hace el conjuro. Se pueden combinar varios efectos sumando sus costes.

| Efecto | Coste | Descripción |
|--------|-------|-------------|
| Daño leve | 1 | Herida menor, molestia |
| Daño moderado | 3 | Herida seria, incapacitante si no se trata |
| Daño grave | 5 | Potencialmente mortal |
| Daño letal | 8 | Muerte si no se evita |
| Penalización leve | 1 | −1 a una habilidad |
| Penalización moderada | 3 | −1 a todas las habilidades, o −2 a una |
| Penalización grave | 5 | −2 a todas o inutilizar una habilidad |
| Curación leve | 1 | Cerrar heridas menores, aliviar dolor |
| Curación moderada | 3 | Sanar heridas graves, curar enfermedad |
| Curación completa | 5 | Restaurar completamente la salud |
| Mejora leve | 1 | +1 a una habilidad |
| Mejora moderada | 3 | +1 a todas o +2 a una |
| Mejora grave | 5 | +2 a todas o +3 a una |
| Control simple | 2 | Una orden concreta a un ser simple (animal, mente débil) |
| Control moderado | 4 | Órdenes complejas o influir en una persona |
| Control total | 7 | Dominio completo de la voluntad |
| Transformación menor | 2 | Cambio cosmético o funcional pequeño |
| Transformación moderada | 5 | Cambio significativo (reemplazar un miembro, alterar un sentido) |
| Transformación mayor | 8 | Cambio radical de naturaleza |
| Creación menor | 3 | Crear algo simple (un objeto pequeño, una criatura básica) |
| Creación mayor | 6 | Crear algo complejo (una criatura inteligente, un objeto elaborado) |
| Protección leve | 1 | Reducir daño o resistir efectos menores |
| Protección moderada | 3 | Bloquear un tipo de daño o efecto |
| Protección completa | 6 | Inmunidad a una categoría de amenaza |
| Conocimiento parcial | 1 | Intuición, pista vaga |
| Conocimiento claro | 3 | Respuesta concreta a una pregunta |
| Conocimiento profundo | 6 | Visión completa de una situación o secreto oculto |

## Aplicación

Cómo llega el efecto al objetivo. Cuanto más fácil de aplicar, más caro.

| Aplicación | Coste | Descripción |
|------------|-------|-------------|
| Ritual directo | 0 | El efecto se aplica durante el propio ritual sobre el objetivo presente |
| Al impactar | 1 | Requiere golpear o tocar al objetivo |
| Al atacar | 3 | Basta con intentar el ataque, aunque falle o se bloquee |
| Proximidad | 4 | Afecta a quien se acerque, sin necesidad de acción |
| Condición | 2 | Se activa cuando se cumple una condición (una palabra, una acción, un evento) |
| Automático | 5 | Se aplica sin acción ni condición, simplemente ocurre |

---

## Alcance

A cuántos objetivos afecta.

| Alcance | Coste | Descripción |
|---------|-------|-------------|
| Uno mismo | 0 | Solo afecta al lanzador |
| Objetivo único | 0 | Un solo objetivo |
| Varios objetivos | 2 | Un grupo pequeño (3-5) |
| Zona | 4 | Todos los que estén en un área |
| Masivo | 7 | Afecta a una multitud o región entera |

---

## Modificadores

Alteran el comportamiento del efecto. Se suman al coste.

| Modificador | Coste | Descripción |
|-------------|-------|-------------|
| Acumulativo | +2 | El efecto se apila con cada aplicación |
| Selectivo | +1 | Puede elegir a quién afecta y a quién no |
| Condicional | −1 | Solo funciona bajo circunstancias específicas (de noche, contra un tipo de criatura...) |
| Retardado | −1 | El efecto no es inmediato, tarda en manifestarse |
| Transferible | +2 | El efecto puede pasar de un objetivo a otro |

## Ejemplos descompuestos

### La espada del naufragio (potencia efectiva 19)

| Componente | Coste |
|------------|-------|
| Penalización leve (−1 a todas las habilidades) | 1 |
| Acumulativo | +2 |
| Al atacar (sin necesidad de impactar) | +3 |
| Efecto secundario: inconsciencia al acumular suficiente | +2 |
| **Total efecto** | **8** |

Potencia sobrante: 11. Margen amplio que la IA puede usar para justificar la intensidad narrativa del arma (el agua fantasmal, la conexión con el naufragio, la dificultad de desencantarla).

### La rata de la mazmorra (potencia efectiva 4)

| Componente | Coste |
|------------|-------|
| Control simple (una orden a un animal) | 2 |
| Ritual directo | 0 |
| Objetivo único | 0 |
| **Total efecto** | **2** |

Potencia sobrante: 2. Margen pequeño pero suficiente para la sencillez del conjuro.

### El brazo del roble (potencia efectiva 7)

| Componente | Coste |
|------------|-------|
| Transformación moderada (reemplazar un miembro) | 5 |
| Mejora leve (+1 fuerza del brazo) | 1 |
| Ritual directo | 0 |
| Objetivo único | 0 |
| **Total efecto** | **6** |

Potencia sobrante: 1. Conjuro ajustado — la curandera usa casi toda su potencia.
